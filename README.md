# Requirements

- [Vue.js](https://github.com/vuejs/vue) `^2.5.2`
- [three](https://github.com/mrdoob/three.js) `^0.115.0`

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
